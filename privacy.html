Política de Privacidade

Esta Política descreve como as informações são tratadas no uso do Contador Pantaneiro.

1. Coleta de informações

O Contador Pantaneiro não coleta nem solicita dados pessoais sensíveis. As interações ocorrem no ambiente da plataforma onde o aplicativo é executado.

2. Uso das informações

O conteúdo informado pelo usuário é utilizado exclusivamente para gerar respostas e orientações técnicas durante a utilização do aplicativo.

3. Armazenamento

O aplicativo não mantém cadastro de usuários nem banco de dados próprio para armazenar conversas. Eventuais registros/telemetria dependem da plataforma e/ou integrações habilitadas pelo usuário.

4. Integrações e bases de conhecimento

Quando habilitado, o aplicativo pode consultar bases de conhecimento autorizadas (ex.: documentos e manuais) para melhorar a qualidade das respostas. O acesso ocorre apenas para fins de recuperação de contexto (RAG).
